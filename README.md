Overview

This project focuses on building an HR Analytics Dashboard to analyze employee data and provide meaningful insights for better human resource decision-making. The dashboard helps organizations understand workforce trends, employee distribution, and attrition patterns using interactive visualizations.

Analyze employee demographics including age, gender, education, and department

Identify attrition trends across job roles, salary slabs, and experience levels
